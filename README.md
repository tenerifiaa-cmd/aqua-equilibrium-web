# Aqua Equilibrium — site vitrine Ring Orion

Site statique d'une seule page (`index.html`, tout embarqué : polices, images, carte).

- `index.html` — le site publié (GitHub Pages).
- `build/` — système de génération : éditer `build/site_template.html` puis exécuter `python3 build/build.py` pour régénérer `index.html`. Ne pas éditer `index.html` à la main.
- Les distributeurs affichés sont des **données de démonstration** — à remplacer dans le tableau `DISTRIBUTEURS` de `build/site_template.html`.
